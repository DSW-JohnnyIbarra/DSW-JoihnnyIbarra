Flex Box

Segun autor (año) El Módulo de Caja Flexible, comúnmente llamado flexbox, fue diseñado como un modelo unidimensional de layout, y como un método que pueda ayudar a distribuir el espacio entre los ítems de una interfaz y mejorar las capacidades de alineación. Este artículo hace un repaso de las principales características de flexbox, las que exploraremos con mayor detalle en el resto de estas guías. Cuando describimos a flexbox como unidimensional destacamos el hecho que flexbox maneja el layout en una sola dimensión a la vez — ya sea como fila o como columna. Esto contrasta con el modelo bidimensional del Grid Layout de CSS, el cual controla columnas y filas a la vez. El eje principal El eje principal está definido por flex-direction, que posee cuatro posibles valores: row row-reverse column column-reverse Si elegimos row o row-reverse, el eje principal correrá a lo largo de la fila según la dirección de la línea.

Al elegir column o column-reverse el eje principal correrá desde el borde superior de la página hasta el final — según la dirección del bloque.

Listas y enlaces

Las listas en HTML nos permiten crear conjuntos de elementos en forma de lista dentro de una página, todos los cuales irán precedidos, generalmente, por un guion o número. Los tipos de listas en HTML son los siguientes: Listas ordenadas Listas desordenadas Listas Ordenadas Las listas en HTML ordenadas son aquellas que nos muestran los elementos de la lista en orden. Para representar el orden tendremos los elementos numerados. Es decir, cada uno de los elementos irá precedido de un número o letra que establezca su orden. Las listas en HTML ordenadas se representan mediante el elemento OL. Cada uno de los elementos de la lista ordenada se representará mediante el elemento LI.

  1. Elemento 1
  2. Elemento 2
  3. Elemento N
Un ejemplo de lista ordenada sería el siguiente:
  1. Julio
  2. Carmen
  3. Ignacio
  4. Elena
Que produciría el siguiente efecto: Julio Carmen Ignacio Elena Número de inicio de la lista: start El atributo start nos permite indicar el número por el cual queremos que empiece la lista, ya que por defecto las listas ordenadas en HTML empiezan por el número 1. De esta forma, si queremos que la lista empiece por el número 5, escribiremos el siguiente código:
  1. Julio
  2. Carmen
  3. Ignacio
  4. Elena
Que produciría el siguiente efecto: 5. Julio 6. Carmen 7. Ignacio 8. Elena
